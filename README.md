# TaskFlow

Eine persönliche Aufgabenverwaltung zum Erfassen von täglichen Todos und zur Übersicht der eigenen Entwicklung über Zeit.

## Was kann die App?

- Tägliche Aufgaben erstellen mit Wiederholung (täglich, wöchentlich, monatlich)
- Aufgaben als erledigt markieren
- Kategorien mit Farb-Labels
- Statistikseite mit Diagramm (pro Tag, Woche, Monat)

## Tech Stack

- TypeScript, HTML, CSS
- Firebase Firestore (Datenspeicherung)
- Firebase Hosting
- esbuild (Bundler)

## Über das Projekt

Dieses Projekt entsteht schrittweise — jeden Tag ein bisschen. Ziel ist es, den Aufbau einer Web-App mit Firebase und TypeScript praktisch zu erkunden und gleichzeitig eine nützliche App für den Alltag zu haben.
